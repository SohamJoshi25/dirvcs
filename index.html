DIRVCS

dirvcs is a command-line version control tool built specifically for directory structures. Unlike Git, it does not track content ‚Äî instead, it tracks and compares snapshots of your folder hierarchy.

‚ú® Features

  • Track directory tree structure over time using snapshots
  • Detect new, deleted, and moved files/folders
  • Export changes as JSON or human-readable tree diffs
  • Print changes with custom indentation
  • Override behavior with config (YAML-based)
  • Rebase, prune, and manage snapshot history
